## Description ##

A wrapper for java.util.concurrent package, ScalaConcurrency wraps a lot of the JDK5,6 and 7 classes to ease their use within Scala code.

  * read/write locks
  * executors
  * scheduling using joda datetime
  * lifecycle for the above
